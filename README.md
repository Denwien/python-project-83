## О проекте

Hexlet Page Analyzer — это веб-приложение для проверки и анализа веб-страниц.  
Позволяет:

- Добавлять URL для анализа.  
- Проверять доступность страниц.  
- Извлекать SEO-данные: `h1`, `title`, `description`.  
- Просматривать историю проверок.  

Проект выполнен с использованием Python, Flask, PostgreSQL и BeautifulSoup.
